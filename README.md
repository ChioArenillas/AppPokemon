# Pokémon App

## 🧩 Description

Pokémon App is a React-based web application that allows users to explore Pokémon data through a clean and interactive interface.

The project focuses on fetching and displaying dynamic data, providing users with an engaging way to browse and interact with Pokémon information.

## 🚀 Live Demo

https://app-pokemon-five.vercel.app/

## 🚀 Technologies

* React
* JavaScript (ES6+)
* CSS
* Vercel (deployment)

## 🎯 Features

* Dynamic Pokémon listing
* Data rendering from external source / API
* Interactive UI for browsing Pokémon
* Conditional rendering based on data
* Responsive design

## 🧠 Key Concepts

* Data fetching and asynchronous logic
* State management with React hooks
* Component-based architecture
* Handling dynamic data
* Separation of logic and presentation

## 💡 Purpose

This project was built to practice working with dynamic data and APIs, as well as improving UI rendering based on real-time information.
